Ardulink

A library for Arduino Simulink comunication.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to ArduLink Page.

Ardulink aims to become a good tool between Arduino and Simulink, helping the community in solving engineering problems to getting data with your microcontroller. The library was made ​​in order to be fully effective without any complicated setup too Simulink. Thus, the user only have two function, the configuration and transmission.

If you would like to view the documentation, see the Doxygen.

Tutorial

Call init function on the setup(). Make sure that all modifications in:
- Terminator.
- Header.
- Baudrate.
- Vector's size.
Vector's format (int32 or single) Need to be modified in Simulink ! The Terminator must be in double quotes:
- "\0","\n","\r\n" and...
The Header must be in single quotes:
- 'L','U','P' and...

Inside the setup, you must call the Init function: 

Ex: Init(9600,8,1);         //By default Terminator is \0 and Header NONE
Ex: Init(9600,3,5,"\0");    //By default Header NONE.
Ex: Init(9600,2,4,"\0",'L');

And inside of the loop the SendBuffer function: 

Ex: SendBuffer(int_vector);
Ex: SendBuffer(float_vector);
Ex: SendBuffer(int_matrix);
Ex: SendBuffer(float_matrix);

If you want to make some declarations:

Ex: int vector[]   = {1, 2, 3, 4, -1, -2, -3, -4};
Ex: float vector[] = {1.3, 2.5, 3.2, 4.7 ,-1.9 ,-2.4 ,-3.6 ,-4.3};
Ex: int matrix[]=
{
  1,  2,  3,  4, 
 -1, -2, -3, -4
}; //matrix [2][4];
Ex: float matrix[]=
{
   1.3,  2.5,  
   3.2,  4.7,
  -1.9 ,-2.4,
  -3.6 ,-4.3
}; //matrix [4][2];

Instalation

  1. Download the Master branch from gitHub.
  2. Unzip.
  3. Paste the folder on your Library folder (On your Libraries folder inside Sketchbooks or Arduino software).
